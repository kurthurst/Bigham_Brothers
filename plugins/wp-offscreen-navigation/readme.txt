== Description ==

**WP Offscreen Navigation** Creates a new theme location where you can add any custom menu. It will then display it on your website on every page and is available via the menu button on the top left side of the browser window.


= Based on =
* The wonderfull [Sidr](www.berriart.com/sidr/) jQuery library

= Website =
http://www.cozmoslabs.com/

== Installation ==

1. Upload the wp-offside-navigation folder to the '/wp-content/plugins/' directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Then navigate to Appareance => Menus and create a new menu. After that asign that menu to the "Offscreen Nav" Theme Location.

== Frequently Asked Questions ==

= How do I display it on my site? =

Navigate to Appareance => Menus and create a new menu. After that asign that menu to the "Offscreen Nav" Theme Location.

= What about browser support? =

All modern browsers except IE7 and older.
